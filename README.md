# Retourenmanagement Tool

Internal prototype of a returns management system
for German small and medium-sized businesses.

## Purpose
Automates and structures return workflows:
- return registration
- status tracking
- refund / replacement logic

## Target market
German SMEs (E-Commerce, Warehouses, Retail)

## Tech stack
- Laravel (API)
- PostgreSQL
- Vue 3 + Vite

## Status
⚠️ Early development / not production ready
